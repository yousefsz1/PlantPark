# Icon source

The PNG icons in this folder are rasterized (SVG → PNG, 128×128) from the
"Flat" style of Microsoft's [Fluent Emoji](https://github.com/microsoft/fluentui-emoji)
set (commit fetched: `main` branch, 2026-07-05).

| File | Fluent Emoji source |
|---|---|
| `water_drop.png` | Droplet / Flat |
| `sun.png` | Sun / Flat |
| `seedling.png` | Seedling / Flat |
| `thermometer.png` | Thermometer / Flat |
| `ruler.png` | Straight ruler / Flat |
| `cherry_blossom.png` | Cherry blossom / Flat |
| `red_apple.png` | Red apple / Flat |
| `house.png` | House / Flat |
| `warning.png` | Warning / Flat |
| `cat_face.png` | Cat face / Flat |
